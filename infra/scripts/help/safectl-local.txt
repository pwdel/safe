Local commands:
  local bootstrap              Run macOS -> Multipass bootstrap + provisioning
  local shell                  Open Multipass shell as default VM user (ubuntu)
  local operator-shell         SSH to VM as operator user
  local runner-shell           Enter coding container shell via safe helper
  local codex-login            Run codex device login inside coding container
  local fork-shell <name>      Enter coding container shell in /workspace/<name>
  local helper <safe-*> [...]  Run /usr/local/bin/safe-* helper on local VM via operator SSH
  local status                 Show VM info + runner status over operator SSH
  local test                   Validate operator access + runner start/status
  local help                   Show this local help

Notes:
  - `local shell` lands on VM host as ubuntu, which may not have direct docker socket access.
  - Use `local operator-shell` for operator-level host access.
  - Use `local runner-shell` for direct container-shell access.

Examples:
  ./safe local bootstrap
  ./safe local operator-shell
  ./safe local runner-shell
  ./safe local codex-login
  ./safe local fork-shell socialpredict
  ./safe local helper safe-rebuild-runner
  ./safe local helper safe-sync-task-spec
  ./safe local helper safe-enter-task-spec
  ./safe local helper safe-run-task-runner --once
